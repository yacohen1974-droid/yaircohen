# **App Name**: Moran Paz Practice

## Core Features:

- Informational Content Display: Showcase pages for home, therapy process, 'BeinMe' workshop, and the Tivon clinic location, providing detailed textual and image content.
- Contact and Inquiry Form: Allows visitors to submit their name, phone, email, and a message to Moran Paz for consultations or inquiries.
- Responsive and Interactive Navigation: Provides intuitive navigation through a main menu and a mobile menu, with visual feedback on scroll for enhanced user experience across all devices.
- Direct Communication Links: Quick access buttons for WhatsApp, email, and direct phone calls to facilitate immediate contact.
- AI-Powered FAQ Assistant: An intelligent tool that can answer common questions about Moran Paz's therapy approaches, workshops, and clinic details by reasoning over the provided site content, offering quick support to prospective clients.

## Style Guidelines:

- Primary color: Muted lavender-blue (#8C78B8) chosen to evoke a sense of thoughtfulness and gentle healing, providing a calming and unique touch against the neutral background.
- Background color: Very light purple-tinted white (#F9F7FF), subtly reflecting the primary hue for a harmonious and soft base, suitable for a serene therapeutic setting.
- Accent color: A rich, calming blue (#336ECC), analogous to the primary hue, used to highlight calls to action and important information, providing good contrast and depth.
- Headlines font: 'Frank Ruhl Libre', a serif font for a classic, literary, and thoughtful aesthetic. Body font: 'Assistant', a clean sans-serif for optimal readability and a modern feel.
- Utilize clean, modern line icons (such as those from Lucide React library) to maintain a minimalist yet communicative visual style throughout the application.
- Maintain a spacious and content-focused layout with consistent maximum content width. Implement a right-to-left (RTL) design to support Hebrew text direction. Utilize responsive grid systems for adaptive content presentation on various screen sizes.
- Incorporate subtle reveal animations for new sections and elements on scroll, along with smooth transitions for navigation and hover states, to create an elegant and engaging user experience.
